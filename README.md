# 🌿 PlantLens - Página de destino conceitual

Esta é a Landing Page do **PlantLens**, um projeto acadêmico desenvolvido com foco na apresentação de um aplicativo voltado para **identificação de plantas através da câmera** e auxílio no cuidado diário.

O site foi criado como uma interface conceitual, simulando a experiência de um produto digital real, com foco em design, usabilidade e apresentação de funcionalidades modernas.

🔗 Acesse o site (Design Responsivo): *adicione seu link aqui*

---

## 🧾 Sobre o Projeto

O **PlantLens** é uma aplicação pensada para facilitar a vida de quem gosta de plantas, permitindo identificar espécies rapidamente e oferecendo suporte completo para manutenção saudável.

A proposta do projeto é unir tecnologia e praticidade, trazendo recursos inteligentes para o dia a dia.

---

## ⚙️ Funcionalidades Apresentadas

* 📸 Identificação de plantas via câmera
* 🌱 Informações detalhadas sobre cada planta
* ⏰ Lembretes e alarmes para cuidados (rega, luz, etc.)
* 📚 Tutoriais e dicas de manutenção
* 🤖 Simulação de uso de inteligência artificial
